# Node.js Express Template

A modern Node.js web application template with Express, development tooling, and best practices.

## Features
- Express.js web framework for HTTP server
- Modern ES6+ JavaScript with proper imports
- ESLint and Prettier for code quality
- Jest for unit testing
- Nodemon for development hot reload
- Docker support for containerization
- Environment configuration with dotenv
- Security best practices with helmet

## Dependencies
- express
- helmet
- dotenv
- cors

## Tools
- ESLint for code linting
- Prettier for code formatting
- Jest for testing framework
- Nodemon for development server
- Docker for containerization

## Setup

```bash
npm install
npm run lint
npm test
npm run dev
```

Start the production server:
```bash
npm start
```
